### **Embed Creator - Code Projects**
Crie mensagens incríveis para o Discord com o Embed Creator! 🚀

Este projeto permite que você crie e envie mensagens personalizadas utilizando **Webhooks do Discord**. Personalize o título, descrição, cores, imagens e muito mais através de uma interface intuitiva e prática.

---

#### **Funcionalidades**
- Interface responsiva e moderna desenvolvida com **TailwindCSS**.
- Personalização completa de **embeds** (cores, thumbnails, imagens de fundo, descrição, footer).
- Integração com Webhooks do Discord para envio rápido de mensagens.
- Suporte a seleção de emojis diretamente no modal de criação.

---

#### **Como Usar**
1. Clone este repositório.
2. Abra o arquivo `index.html` no navegador.
3. Insira as informações no formulário, incluindo o Webhook do Discord.
4. Clique em "Enviar Mensagem" para ver o resultado no seu canal do Discord.
